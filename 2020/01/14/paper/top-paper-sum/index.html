论文总汇

置顶 | Posted on 2020-01-14 | In 论文
| Words count in article 2.1k | Reading time 8

置顶文章总结每篇论文的主要内容

1. Task Offloading for Mobile Edge Computing in Software Defined Ultra-Dense Network

移动边缘计算在软件定义超密集网络中的任务卸载

本人边缘计算入门的第一篇英文文献

期刊：IEEE JOURNAL ON SELECTED AREAS IN COMMUNICATIONS 1区

作者：Min Chen等

提出了软件定义额超密集网络架构（SD-UDN），充分考虑了多服务器和超密集网络的环境

2. 基于边缘计算的交通信号灯联合控制

看的第一篇边缘计算结合交通的论文

北邮的汪煜超的硕士毕业论文

引入边缘计算和强化学习进行交叉口的交通控制

3. Enhancing the User Experience in Vehicular Edge Computing Networks-An Adaptive Resource Allocation Approach

增强车辆边缘计算网络中的用户体验：一种自适应资源分配方法

偏数学理论

期刊：IEEE ACCESS 2区但是被吐糟的很厉害的期刊

作者：XIAOKE SUN等

设计了一种自适应资源分配方法；利用Lyapunov优化技术，将问题分解为一系列的时隙优化问题，并使用DC规划解之。

4. In-Edge AI :Intelligentizing Mobile Edge Computing, Caching and Communication by Federated Learning

通过联合学习对移动边缘计算，缓存和通信进行智能化

看的边缘计算结合联邦学习的第一篇论文

期刊：IEEE Network 1区

作者：Xiaofei Wang等

利用联邦学习在训练DRL Agent，DRL在边缘端做缓存的决策，在UE做任务卸载的决策，Double DQN，使用Xender的数据集

5. Cooperative adaptive driving for platooning autonomous self driving based on edge computing

基于边缘计算和车辆队列的自动驾驶

期刊：INTERNATIONAL JOURNAL OF APPLIED MATHEMATICS AND COMPUTER SCIENC 4区

作者：BEN-JYE CHANG等

感觉东西写一大堆，没想到是个4区的

将车辆划分为一个个排（队列），以排为单位（排头等）自适应驾驶

6. Adaptive Federated Learning in Resource Constrained Edge Computing Systems

在资源受限的边缘计算系统中进行自适应联合学习

偏理论

期刊：IEEE JOURNAL ON SELECTED AREAS IN COMMUNICATIONS 1区

作者：Shiqiang Wang等

聚合器的聚合频率动态决定

7. QoE-Driven Content-Centric Caching With Deep Reinforcement Learning in Edge-Enabled IoT

基于QoE驱动的以内容为中心的缓存以及基于边缘物联网中的深度强化学习

期刊：IEEE COMPUTATIONAL INTELLIGENCE MAGAZINE 1区

作者：Xiaoming He等

感觉写的一般…

先关于QoE建模，再使用FEL-DRL （fixed target network (F), experience replay buffer (E), and adaptive learning rate (L)）, DQL

8. Optimal and Scalable Caching for 5G Using Reinforcement Learning of Space-Time Popularities

使用基于时空流行度的强化学习实现5G的最优可扩展缓存

期刊：IEEE JOURNAL OF SELECTED TOPICS IN SIGNAL PROCESSING 1区

作者：Alireza Sadeghi等

基于内容的动态时空流行度分为全局和局部建模，使用approximation Q learning逼近Q值，减少了损失函数的参数从而减少Q表大小。可以用Deep Q learning做

9. Integrated Networking, Caching, and Computing for Connected Vehicles: A Deep Reinforcement Learning Approach

集成网络、缓存和计算的互联车辆：一种深度强化学习方法

期刊：IEEE TRANSACTIONS ON VEHICULAR TECHNOLOGY 2区

作者：Ying He等

把网络、缓存和计算能力整合在一起考虑，使用Double-Dueling-Deep Q-network。网络方面不熟悉，可以考虑能耗。

10. Edge Intelligence: Paving the Last Mile of Artificial Intelligence With Edge Computing

边缘智能：用边缘计算铺就人工智能的最后一公里 综述

期刊：PROCEEDINGS OF THE IEEE 1区

作者：ZHI ZHOU等

11. Smart caching based on user behavior for mobile edge computing

移动边缘计算中基于用户行为的智能缓存

期刊：Information Sciences 2区

作者：Yuanyuan Zeng等

使用卷积神经网络提取用户偏好上下文

，再根据缩放算法求解Top缓存。

12. ECDU: an edge content delivery and update framework in Mobile edge computing

期刊：EURASIP JOURNAL ON WIRELESS COMMUNICATIONS AND NETWORKING 4区

作者：Chuntao Ding等

这就是4区的水平吗。。。

13. Q-learning based collaborative cache allocation in mobile edge computing

期刊：Future Generation Computer Systems 2020 1区

作者：Wei-Che Chien等

在C-RAN和SDN架构下多RRH和多BBU，提出了Q-LCCA来解决协同缓存问题，并定义UCCHE（单元缓存容量命中效率）来度量缓存性能，并根据提出的RAP策略选择全局或局部缓存动作。5G下的C-RAN架构现在已经演变成了RRH + BBU（CU + DU），其中BBU被进一步划分为CU和DU

14. Deep Reinforcement Learning for Cooperative Content Caching in Vehicular Edge Computing and Networks

期刊：IEEE INTERNET OF THINGS JOURNAL 1区

作者：Guanhua Qiao等

关于缓存问题提出了双时标马尔可夫模型（DTS-MDP），内容及时性的变化没有车辆移动性和网络状态那么频繁。使用DDPG解。

15. Artificial Intelligence Empowered Edge Computing and Caching for Internet of Vehicles

期刊：IEEE Wireless Communications 1区

作者：Yueyue Dai等

因为EI需要的大量历史数据来进行准确的预测，同时又要缓存流行内容，所以本文使用DDPG来权衡计算任务和缓存。感觉就是写写，没什么实质的东西。

16. Distributed Deep Learning at the Edge: A Novel Proactive and Cooperative Caching Framework for Mobile Edge Networks

期刊：IEEE WIRELESS COMMUNICATIONS LETTERS 本来是2区，2020被降为3区

作者：Yuris Mulya Saputra等

提出了两种缓存架构，1. 中心服务器从边缘收集信息然后执行深度学习算法预测内容流行度。由于会泄露隐私因此进一步提出2.分布式训练DL算法，中心服务器起聚合作用。感觉写的一般啊。

17. A Q-Learning-Based Proactive Caching Strategy for Non-Safety Related Services in Vehicular Networks

期刊：IEEE INTERNET OF THINGS JOURNAL 1区

作者：Lu Hou等

使用LSTM预测车辆行驶方向，使用Q learning 最大化奖励传输延迟。他这里好像是一个车辆只请求一个content啊，写的不是很明白，他的数据集是车辆移动性数据集。

18. A data-driven approach of performance evaluation for cache server groups in content delivery network

期刊：JOURNAL OF PARALLEL AND DISTRIBUTED COMPUTING 3区

作者：Ziyan Wua等

使用特征工程+LSTM auto-encoding 预测请求到达率

19. Localized Small Cell Caching: A Machine Learning Approach Based on Rating Data

期刊：IEEE TRANSACTIONS ON COMMUNICATIONS 2区

作者：Peng Cheng等

使用贝叶斯概率矩阵分解预测个人内容流行度，再用强化学习（DCA）决策

20. A Deep Learning Model Generation Framework for Virtualized Multi-Access Edge Cache Management

期刊：IEEE ACCESS 2区

作者：KYI THAR等

使用强化学习（Q learning）来选择预测内容流行度的神经网络（CNN, RNN等）

21. Online Proactive Caching in Mobile Edge Computing Using Bidirectional Deep Recurrent Neural Network

期刊：IEEE INTERNET OF THINGS JOURNAL 1区

作者：Laha Ale等

使用 Conv + 双向LSTM 预测内容访问频率。在该文中，内容使用one-hot编码，使用卷积层减少编码后的信息，再使用双向LSTM预测访问频率。

21. Using Grouped Linear Prediction and Accelerated Reinforcement Learning for Online Content Caching

会议：IEEE International Conference on Communications (ICC) / Workshop on Integrating UAVs into 5G

C类会议

作者：Naifu Zhang等

这篇文章，使用GLM预测内容流行度，在有限的样本情况下自己模拟样本生成来加速强化学习。

22. Deep Reinforcement Learning-Based Video Quality Selection and Radio Bearer Control for Mobile Edge Computing Supported Short Video Applications

期刊：IEEE ACCESS

作者：WENJUN WU等

基于深度强化学习的视频质量选择和无线电承载控制，用于移动边缘计算支持的短视频应用。使用PG-QR，其实就是policy gradient算法。。。联合radio bearer和视频质量的控制

23. Caching Transient Data for Internet of Things: A Deep Reinforcement Learning Approach

24. DeepCachNet A Proactive Caching Framework Based on Deep Learning in Cellular Networks

25. A Deep Reinforcement Learning-Based Framework for Content Caching

会议：Conference on Information Sciences and Systems (CISS)，不知道是什么级别的会议

作者：Chen Zhong等

可能是使用深度强化学习来解决缓存问题的鼻祖了，他自己是这么说的。

26. Energy Minimization in D2D-Assisted Cache-Enabled Internet of Things A Deep Reinforcement Learning Approach

期刊：IEEE TRANSACTIONS ON INDUSTRIAL INFORMATICS 1区

作者：Jie Tang等

着眼于D2D网络下的缓存来减少能量损耗，分别在SBS和UE上运行DQN和QN，这篇文章的时隙长度可变，使用固定的请求数量来表示一个时隙

27. Convergence of Networking and CloudEdge Computing Status, Challenges, and Opportunities

期刊：IEEE NETWORK 1区

作者：Qiang Duan等

偏工程的云边概述

28. Deep Reinforcement Learning for Backscatter-Aided Data Offloading in Mobile Edge Computing
