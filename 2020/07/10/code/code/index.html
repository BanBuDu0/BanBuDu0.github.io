动态规划

动态规划算法实质上是用空间换时间的算法，使用能接受的额外空间来提高运行效率。

使用动态规划必须满足两个条件：

  1. 无后效性

    或者可以称其为马尔可夫性质，即当前的状态只和当前状态的前一个状态有关，而与前一个状态之前的所有状态都无关。可以说某一状态是其之前所有状态的总结。

  2. 最优子结构

    如果一个问题最优的解决方案可以通过最优它的子问题的解决方案来获取，那么这个问题就用最优子结构的属性
