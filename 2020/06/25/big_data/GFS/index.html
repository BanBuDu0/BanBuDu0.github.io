Why is distributed storage hard

  1. 高性能(high performance) -> 数据分发到多个服务器上
  2. 很多服务器 -> 容错能力
  3. 容错(fault tolerance) -> 复制
  4. 复制(replication) -> 一致性问题
  5. 更好的一致性(consistency) -> 性能下降
