2. 在数值优化中，一般采用迭代法求解无约束优化问题min f(x)的极小点，请描述一下无约束优化问题的一般算法框架

  1. 给定初始化参数以及初始迭代点 $x_0$. 置 $k:= 0$
  2. 若 $x_k$满足某种终止准则，停止迭代，以$x_k$作为近似极小
  3. 通过求解$x_k$处的某个子问题确定下降方向$d_k$
  4. 通过某种搜索方式确定步长因子$\alpha_k$，使得$f(x_k+\alpha_kd_k) < f(x_k)$
  5. 令$x_{k+1}:=x_k + \alpha_k d_k, k:=k+1$，转2

3. 描述求解无约束最优化问题的牛顿法步骤

  1. 给定终止误差值$0 \le \epsilon << 1$，初始点$x_0 \in \mathbb R^n$. 令 $k := 0$.
  2. 计算 $g_k = \nabla f(x_k)$. 若 $||g_k|| \le \epsilon$，停算，输出 .
  3. 计算$G_k = \nabla^2f(x_k)$，并求解线性方程组得解 $d_k$，
  4. 令 $x_{k+1} = x_k + d_k, k := k+1$，转2.

8. 设二次规划其中H为n阶对称正定矩阵，矩阵A行满秩，求其最优解并说明解的唯一性

首先构造拉格朗日函数：$L(x,\lambda) = \frac12x^THx+c^Tx - \lambda^T(Ax-b)$，

$\frac{\partial L(x,\lambda)}{\partial x} = Hx + c - A^T\lambda = 0$,

两边乘以$H^{-1}\quad-> \quad x + H^{-1}c - H^{-1}A^T\lambda = 0$,
两边乘以$A \quad -> \quad Ax + AH^{-1}c - AH^{-1}A^T\lambda = 0$,
$ \quad -> \quad b + AH^{-1}c - AH^{-1}A^T\lambda = 0$,
$ \quad -> \quad b + AH^{-1}c = AH^{-1}A^T\lambda$,
$ \quad -> \quad \lambda = (AH^{-1}A^T)^{-1}(b + AH^{-1}c)$,
$ \quad -> \quad x = H^{-1}A^T(AH^{-1}A^T)^{-1}(b + AH^{-1}c) - H^{-1}c$,
解的唯一性：因为$H$正定，$A$行满秩；所以拉格朗日矩阵非奇异，即其解唯一。

设A是m*n实满秩矩阵，a为n维实向量，求解二次规划以及相应的拉格朗日乘子

首先构造拉格朗日函数：$L(x,\lambda) = \frac12x^THx-a^Tx + \frac12a^Ta - \lambda^T(Ax-b)$，其中$H$是单位阵

$\frac{\partial L(x,\lambda)}{\partial x} = Hx -a - A^T\lambda = 0$,

两边乘以$H^{-1}\quad-> \quad x - H^{-1}a - H^{-1}A^T\lambda = 0$,
两边乘以$A \quad -> \quad Ax - AH^{-1}a - AH^{-1}A^T\lambda = 0$,
$ \quad -> \quad b - AH^{-1}a - AH^{-1}A^T\lambda = 0$,
$ \quad -> \quad b - Aa - AA^T\lambda = 0$,
$ \quad -> \quad \lambda=(AA^T)^{-1}(b - Aa)$,
$ \quad -> \quad x=a + A^T(AA^T)^{-1}(b - Aa)$.
